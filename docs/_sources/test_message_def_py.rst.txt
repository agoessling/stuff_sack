Functions
^^^^^^^^^

.. py:function:: unpack_message(buf)

  Unpack buffer into message instance based on header UID.

  :param bytes-like buf: an array to be unpacked.
  :return: a message instance.
  :raises UnknownMessage: UID from buffer header is unknown.
  :raises Various: See individual :py:meth:`unpack` methods.

Exceptions
^^^^^^^^^^

.. py:exception:: UnpackError

  General unpack error.

.. py:exception:: UnknownMessage

  **Bases:** :py:exc:`UnpackError`

  Unknown UID in buffer header.

.. py:exception:: IncorrectBufferSize

  **Bases:** :py:exc:`UnpackError`

  Buffer has incorrect length for message.

.. py:exception:: InvalidUid

  **Bases:** :py:exc:`UnpackError`

  Buffer header has incorrect UID for message.

.. py:exception:: InvalidLen

  **Bases:** :py:exc:`UnpackError`

  Buffer header has incorrect length for message.
